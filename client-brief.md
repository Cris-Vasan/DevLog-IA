# Client Brief — DevLog AI

## 1. Nombre del proyecto
DevLog AI

## 2. Descripción general
DevLog AI es una aplicación web full stack pensada para desarrolladores y estudiantes de software. Su objetivo es ayudar a registrar tareas técnicas, sesiones de trabajo, avances del proyecto y convertir notas de desarrollo en tareas o issues más estructurados.

## 3. Problema
Durante el desarrollo de software, muchas ideas, pendientes, errores y avances quedan dispersos entre notas personales, mensajes, commits y documentos. Esto dificulta hacer seguimiento del trabajo, priorizar tareas y documentar el progreso real del proyecto.

## 4. Objetivo principal
Construir una aplicación útil para el día a día de un desarrollador que permita:
- Registrar tareas técnicas
- Guardar sesiones de trabajo
- Visualizar el estado del proyecto
- Convertir notas breves en tareas o issues más claros
- Mantener una trazabilidad básica del avance del desarrollo

## 5. Enfoque técnico
El proyecto tendrá frontend y backend, y se desarrollará usando Claude Code como herramienta principal. Como base tecnológica, se priorizará el uso de Node.js y React, ya que son tecnologías con las que se tiene mayor experiencia.
